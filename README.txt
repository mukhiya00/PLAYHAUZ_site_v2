# PlayHauz Website (Static)

## Run locally
Just open `index.html` in your browser.

## Deploy
Upload the folder to:
- Netlify (drag & drop)
- GitHub Pages
- Render Static Site

## Edit
- Images: `assets/img/`
- Styles: `assets/style.css`
- Booking steps logic: `assets/app.js`
- Pages: `index.html`, `services.html`, `gallery.html`, `booking.html`
